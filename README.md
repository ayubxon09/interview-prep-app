## 🚀 APK Olish — Bosqichma-bosqich

### 1-qadam: GitHub akkaunt oching
- [github.com](https://github.com) ga kiring
- "Sign up" bosib ro'yxatdan o'ting (bepul)

### 2-qadam: Yangi repo oching
- GitHub'da `+` → "New repository" bosing
- Nom: `interview-prep-app`
- **Public** tanlang
- "Create repository" bosing

### 3-qadam: Fayllarni yuklang
- Repo sahifasida "uploading an existing file" bosing
- Ushbu ZIP faylni oching va BARCHA fayllarni yuklang
- "Commit changes" bosing

### 4-qadam: APK avtomatik build bo'ladi!
- GitHub "Actions" tabiga o'ting
- "Build Flutter APK" workflow ishga tushadi (~5-7 daqiqa)
- Tugagach, workflow nomini bosing
- "Artifacts" bo'limida **interview-prep-app** ni yuklab oling

### 5-qadam: Telefoningizga o'rnating
- Yuklangan ZIP ni oching → `app-release.apk` ni toping
- Telefoningizga o'tkazing (Telegram orqali o'zingizga yuboring)
- Telefoningizda oching
- "Noma'lum manbadan o'rnatishga ruxsat" bering
- O'rnating ✅
